## Deskripsi Program 
Repositori ini berisi implementasi forward propagation CNN, RNN, dan LSTM from scratch. Bagian CNN dikerjakan dalam konteks task image classification menggunakan dataset Intel Image Classification. Bagian RNN dan LSTM dikerjakan dalam konteks task image captioning menggunakan dataset Flickr8k. Kedua arsitektur (RNN dan LSTM) dilatih secara terpisah sebagai decoder dan dibandingkan hasilnya.
